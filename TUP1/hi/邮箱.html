QQ邮箱SMTP前端工具

使用SMTP.js直接连接QQ邮箱服务器

安全警告

此工具仅用于演示目的。在前端JavaScript中硬编码邮箱授权码存在严重安全风险，任何访问您网站的用户都可以查看您的授权码，可能导致您的邮箱被滥用。生产环境中请务必使用后端服务处理邮件发送。

注意：需要在QQ邮箱设置中生成SMTP授权码

推荐的安全替代方案

在生产环境中，建议使用后端API来处理邮件发送：

// 前端代码 - 发送请求到后端API fetch('/api/send-email', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ to: 'recipient@example.com', subject: '邮件主题', message: '邮件内容' }) }) .then(response => response.json()) .then(data => { if (data.success) { alert('邮件发送成功！'); } else { alert('邮件发送失败：' + data.error); } });

后端服务（例如Node.js）可以安全地存储和使用SMTP凭据：

// 后端代码 - Node.js示例 const nodemailer = require('nodemailer'); app.post('/api/send-email', async (req, res) => { const { to, subject, message } = req.body; const transporter = nodemailer.createTransport({ host: 'smtp.qq.com', port: 587, secure: false, auth: { user: process.env.QQ_EMAIL, // 从环境变量读取 pass: process.env.QQ_PASSWORD // 从环境变量读取 } }); try { await transporter.sendMail({ from: process.env.QQ_EMAIL, to, subject, text: message }); res.json({ success: true }); } catch (error) { res.json({ success: false, error: error.message }); } });
